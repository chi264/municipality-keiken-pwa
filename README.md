# 市区町村経県値PWA

スマホで使うことを前提に作り直した、市区町村版の経県値アプリです。

## できること

- 都道府県ごとの達成状況を見る
- 市区町村ごとにステータスを登録する
- ステータスは `未訪問 / 通過 / 降りた / 観光 / 宿泊 / 居住`
- 訪問日とメモを残す
- スマホ本体に保存する
- JSONでバックアップ/復元する
- PWAとしてホーム画面に追加する

## 今のデータについて

都道府県の市区町村総数は達成率の分母として入れています。
ただし、最初の版では各都道府県の代表的な市区町村だけを収録しています。

次の段階で、全市区町村データを `app.js` の `sampleMunicipalities` に追加すれば、全国版として拡張できます。

## スマホで普段使いするなら

最終的には GitHub Pages / Vercel / Netlify などに置いて、HTTPSのURLで開くのがおすすめです。

```text
https://your-app.example/
```

そのURLをスマホで開き、「ホーム画面に追加」すると普通のアプリに近い使い方になります。

PCとの同期はしない前提なので、データはスマホ内に保存されます。
機種変更やバックアップにはJSON書き出しを使ってください。
