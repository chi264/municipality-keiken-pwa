# 旅市区町村メモPWA

スマホで使うことを前提にした、市区町村版の経県値アプリです。

## できること

- 都道府県ごとの達成状況を見る
- 市区町村カードにステータス色を付ける
- 全市区町村データを最初から持つ
- 市区町村ごとに `未訪問 / 通過した / 乗り換えた / 降りた / 観光した / 宿泊した / 住んだ` を記録する
- 訪問日、タグ、また行きたい度、メモを残す
- 北海道・関東中心の鉄路から、出発・到着を選んで通過市区町村をまとめて登録する
- 通過候補を自分でチェック解除してから登録する
- 鉄路候補に抜けている市区町村を手で追加してから登録する
- 旅行単位のメモを残す
- 都道府県別の制覇率と次に埋める候補を見る
- JSONでバックアップ・復元する

## データについて

市区町村データは `municipalities-data.js` に収録しています。政令指定都市の区・特別区も含めた「市区町村」寄りのデータです。

鉄路データはまず北海道・関東を中心に、よく使いそうな路線を手入力のプリセットとして入れています。実際の経路と完全一致しない場合は、登録前に候補チェックを外したり、市区町村を追加したりして調整できます。

## スマホで使う

GitHub PagesなどのHTTPS URLで開き、ブラウザメニューからホーム画面に追加します。記録はPCとは同期せず、そのスマホのブラウザ内に保存されます。
