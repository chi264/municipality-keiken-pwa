# 旅市区町村メモPWA

スマホで使うことを前提にした、旅行メモ・市区町村経県値・鉄道通過ログのPWAです。

## できること

- 都道府県ごとの達成状況を見る
- 市区町村ごとにステータスを登録する
- ステータスは `未訪問 / 通過した / 乗り換えた / 降りた / 観光した / 宿泊した / 住んだ`
- 訪問日とメモを残す
- タグやまた行きたい度を残す
- 鉄道路線・区間から通過市区町村をまとめて登録する
- 旅行単位のメモを残す
- あと少しで埋まる都道府県を提案する
- スマホ本体に保存する
- JSONでバックアップ/復元する
- PWAとしてホーム画面に追加する

## 今のデータについて

都道府県の市区町村総数は達成率の分母として入れています。
ただし、最初の版では各都道府県の代表的な市区町村だけを収録しています。

次の段階で、全市区町村データを `app.js` の `sampleMunicipalities` に追加すれば、全国版として拡張できます。

## スマホで普段使いするなら

GitHub PagesのURLをスマホで開き、「ホーム画面に追加」すると普通のアプリに近い使い方になります。

PCとの同期はしない前提なので、データはスマホ内に保存されます。
機種変更やバックアップにはJSON書き出しを使ってください。
